Chronic Kidney Disease


Following are the conditions for Chronic Kidney Disease:

    If Creatinine value > 1.2 on both current and previous encounter values:
    • Stage 1: if on current encounter GFR >= 90
    • Stage 2: if on current encounter 60 < GFR < 89
    • Stage 3: if on current encounter 30 < GFR < 59
    • Stage 4: if on current encounter 15 < GFR < 29
    • Stage 3: if on current encounter GFR < 15
